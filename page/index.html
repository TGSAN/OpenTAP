OpenTAP 是什么

OpenTAP 全称 CloudMoe Open-Source TAP，是一款 CloudMoe 旗下以 SSTap 为原型利用虚拟网卡技术在网络层实现的代理工具。
OpenTAP 通过路由表，能在网络层拦截路由表规则中的所有连接并转发给 Socks5、Shadowsocks、V2Ray 代理，
而无需对被代理的应用程序做任何更改或设置。
她能同时转发 TCP、UDP 数据包，因为 OpenTAP 专为游戏玩家打造，所以
必须连接的节点支持 UDP 才能开启代理。
code

代码开源 *


OpenTAP 使用 GNU GPLv3 进行开源，任何人都可以监管、参与 OpenTAP 项目的开发。

* 开源的版本为 OpenTAP 的社区版（Community），部分私有、实验性功能会在 OpenTAP-Pro 项目中以 Apache License, Version 2.0 协议进行半闭源开发（即特约成员可参与开发、对 OpenTAP-Pro 进行 OEM 定制），OpenTAP-Pro 将不定期合并进社区版分支。

vpn_key

使用虚拟网卡技术


使用 OpenVPN Windows TAP V9 Driver 官方社区版虚拟网卡技术进行开发，稳定性高于三方虚拟网卡驱动，速度可达100Mbps*，并支持根据规则进行局部代理或系统全局代理。

* OpenVPN Windows TAP V9 社区版驱动最高支持100Mbps，并非 OpenTAP 的最大支持速度。
